Dal - Programming Learning Platform

Description:
Dal is a frontend-based web platform designed to help beginners learn programming fundamentals in a structured and interactive way. The website provides essential programming resources, learning roadmaps, and practical exercises to guide users through their journey. It focuses on making programming more accessible through a simple and engaging UI.

Technologies Used:
- HTML, CSS, JavaScript: Core structure and interactivity
- Bootstrap: For responsive design and UI components
- Git & GitHub: For version control and deployment
