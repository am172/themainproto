BlogApp - A Simple Blogging Platform

Description:
BlogApp is a full-stack blogging platform built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js). Users can create an account, log in securely, and write blog posts with images.

Technologies Used:
- React.js: Dynamic and interactive UI.
- Tailwind CSS: Modern styling with responsive design.
- Node.js & Express.js: Backend API for user authentication and blog management.
- MongoDB: NoSQL database for storing users and blog posts.
- JWT Authentication: Secure login system.
- RESTful API: Efficient communication between frontend and backend.
- Git & GitHub: Version control and project management.
